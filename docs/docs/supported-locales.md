# Supported Locales

89 locales are bundled. Retrieve them at runtime with:

```typescript
RuleBasedNumberFormat.getSupportedLocales();
```

:::note
The `und` locale contains historic numeral systems (Roman, Greek, Hebrew, Armenian, and more).
See [The `und` Locale](./und-locale) for details.
:::
